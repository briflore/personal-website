Color Guesser

iOS Mobile Application

Color Guesser is an interactive mobile game designed for iOS that challenges users to identify colors based on their hex codes or RGB values. It blends educational elements with a fast-paced gaming experience.

The project was an exploration into mobile UI/UX design and swift development. The goal was to create an interface that felt light, responsive, and aesthetically pleasing, mirroring the very colors the user is trying to guess.
